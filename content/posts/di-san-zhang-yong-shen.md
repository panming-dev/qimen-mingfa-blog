用神，是奇门遁甲中**代表所问事物的那个符号**。问事业看开门，问财运看生门，问婚姻看六合——这就是用神。

但奇门的用神不是"一个符号打天下"。同一件事，你可以从多个角度切入：问工作，既看开门（事业），也看时干（你自己），还看值符（领导或大环境）。

本章讲解奇门用神的三层体系：核心用神、辅助用神、参考用神。你会学到：八门各自对应什么领域、不同事项的快速查表法、多事同占时的处理技巧。掌握用神，就等于拿到了奇门解盘的"钥匙"——知道该打开哪个"锁"。

## 一、什么是用神？

简单说，用神就是**代表所问事项的符号**。

比如：
- 问事业能否升职？→ 看**开门**落在哪宫
- 问今年财运如何？→ 看**生门**落在哪宫
- 问婚姻感情？→ 看**六合**落在哪宫
- 问身体健康？→ 看**天芮星**落在哪宫

**关键：** 用神选对了，分析方向才对；用神选错了，再会解盘也白搭。

## 二、用神的三层体系

一个事项往往涉及多个层面，需要用神组合来看。

### 1. 核心用神（主看）

**定义：** 最直接代表所问事项的符号。

| 事项 | 核心用神 | 为什么 |
|------|----------|--------|
| 事业、工作、官职 | 开门 + 天心 | 开门主事业，天心主管理 |
| 财运、生意、利润 | 生门 + 天任 | 生门主生长，天任主担当 |
| 婚姻、合作、伴侣 | 六合 + 乙/庚 | 六合主合，乙庚为夫妻 |
| 健康、疾病、学业 | 天芮星 | 天芮主问题、学生 |
| 诉讼、口舌、是非 | 惊门 + 白虎 | 惊门主口舌，白虎主凶 |
| 出行、旅行、变动 | 开门 + 马星 | 开门主出行，马星主动 |
| 寻物、失盗、丢失 | 伤门 + 杜门 | 伤门主损失，杜门主隐藏 |
| 求学、考试、文书 | 辅星 + 景门 + 丁 | 辅星主学业，景门主文书 |

### 2. 辅助用神（参看）

**定义：** 辅助判断的符号，提供补充信息。

| 事项 | 辅助用神 | 作用 |
|------|----------|------|
| 所有事项 | 时干 | 代表你自己、当前状态 |
| 所有事项 | 值符 | 代表主导力量、贵人 |
| 事业 | 日干 | 代表你的能力 |
| 财运 | 六合 | 代表合作伙伴 |
| 婚姻 | 乙（女方）、庚（男方） | 代表双方 |
| 健康 | 病门（天芮）+ 死门 | 代表疾病状态 |
| 诉讼 | 值符（原告）+ 值使（被告） | 代表双方 |

### 3. 参考用神（兼看）

**定义：** 特定情况下的补充视角。

- 寻物：看杜门（隐藏处）、玄武（盗贼）
- 出行：看马星位置、开门是否被冲
- 诉讼：看六合（调解）、惊门（口舌）
- 合作：看六合状态、乙庚关系

## 五、多事同占怎么处理？

**问题：** 一个人同时问几件事，用神怎么选？

**原则：**
1. **一事一问**：最准。同一时间只聚焦一个事项。
2. **时干为基准**：时干宫的状态，是你整体的运势底色。
3. **分宫看**：不同事项看不同宫，不混在一起。

**举例：** 一人同时问事业和婚姻
- 事业：看**开门**宫
- 婚姻：看**六合**宫
- 自己：看**时干**宫
- 分别分析，不混在一起

## 六、用神空亡与入墓

### 1. 空亡

**定义：** 地支轮空，能量"不在场"。

**查法：** 根据旬首查空亡（如甲子旬，戌亥空）。

**含义：**
- 用神空亡 → 这件事不成、落空
- 出空日（如戌亥空，遇戌亥日）可能成

### 2. 入墓

**定义：** 被"关"在库里，能量被困。

**查法：**
- 戊入戌墓（天盘戊+地盘戌）
- 丁入戌墓（天盘丁+地盘戌）
- 丙入戌墓（天盘丙+地盘戌）
- 壬癸入辰墓（天盘壬癸+地盘辰）

**含义：**
- 用神入墓 → 事被卡住、难推进
- 冲墓（辰戌冲）或出墓（遇冲）可解

## 八、用神与时间的关系

### 1. 时干为"我"

**规则：** 时干宫代表**求测者本人**的状态。

**判断：**
- 用神生时干 → 我为这事付出多
- 时干生用神 → 这事对我有利
- 用神克时干 → 这事压制我
- 时干克用神 → 我能控制这事

### 2. 日干为"自身"

如果是当日多事，还可以看日干。

### 3. 年命为"流年运势"

结合求测者的出生年命，看共振。

## 九、实战案例

### 案例1：问事业发展

**求测：** 今年能否升职？

**起盘：** 开门在震三宫，天冲星，伤门

**分析：**
1. 核心用神：开门在震三（旺，震为动）
2. 辅助看：时干在巽四（生开门）
3. 格局：进茹格（三进宫）
4. 星门：天冲+伤门 → 主动但易受伤

**结论：** 事业有上升趋势（开门旺+进茹格），但过程中有冲突（伤门），需防冲动（天冲）。时干生开门，对你有利，可争取。

### 案例2：问婚姻感情

**求测：** 我和TA能结婚吗？

**起盘：** 六合在兑七，乙在震三，庚在坎一

**分析：**
1. 核心：六合（婚姻）在兑七（胎，未旺）
2. 乙（女方）在震三（旺）
3. 庚（男方）在坎一（休）
4. 乙庚关系：乙+庚（日奇被刑）→ 有摩擦
5. 六合状态：乙+丁（奇仪顺遂）→ 有转机

**结论：** 六合落宫虽休，但乙庚有合（震三克坎一，木克土，女方主动），且有丁奇化解。可成，但需女方多主动。

## 本章小结

**用神是奇门解盘的"钥匙"。** 选对钥匙，才能打开正确的锁。

**关键收获：**
1. 不同事项看不同门（开、生、六合...）
2. 三层用神体系：核心 + 辅助 + 参考
3. 空亡入墓要特别注意
4. 用神旺衰决定能量大小

**学习建议：**
1. 把用神速查表背熟
2. 每次解盘先问自己：这事该看哪个门？
3. 多练习"一事一问"的聚焦训练
