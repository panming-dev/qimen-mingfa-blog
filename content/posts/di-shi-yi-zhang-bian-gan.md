遍干，就是天干在九宫中的**流转轨迹**。知道天干从哪里来、到哪里去，才能看出事物的发展脉络。

<p>本章讲解遍干的三条基本定律：1）甲随符行——甲干永远跟着值符走；2）阳顺阴逆——阳遁顺飞，阴遁逆飞；3）六仪三奇序——戊己庚辛壬癸、甲乙丙丁的固定顺序。基于这三点，衍生出：暗干暗支（隐藏的流转信息）、旺动条件（要素何时被激活）、六亲定位（父母兄弟妻子子的宫位）。学会遍干，你就能追踪任何用神的前世今生。</p>

## 一、遍干三定律

### 1. 甲随符行

**定律：** 甲干（及六仪三奇）的流转，永远跟随值符。

**含义：** 值符走到哪，甲（及六仪三奇）就跟到哪。这是飞盘的基础。

**例：** 阳四局，值符在震三，则甲子戊从震三开始顺飞。

### 2. 阳顺阴逆

**定律：** 阳遁顺飞，阴遁逆飞。

**含义：** 能量流动方向由阴阳决定。

**口诀：** 阳顺阴逆别混淆，夏至之后用阴遁。

### 3. 六仪三奇序

**定律：** 飞布顺序固定：
- 六仪：戊 → 己 → 庚 → 辛 → 壬 → 癸
- 三奇：乙 → 丙 → 丁

**含义：** 无论从哪个宫起，顺序不变。

## 二、暗干暗支

### 什么是暗干暗支？

每个宫除了表面的天盘干、地盘干，还有**隐藏的流转信息**：
- **天盘暗干**：天盘干从哪宫来？
- **地盘暗支**：地盘支向哪宫去？

### 暗干查法

暗干就是值符飞布时携带的天干序列。

**阳遁暗干序列（从值符宫起）：**
```
戊、己、庚、辛、壬、癸、乙、丙、丁
```

**阴遁暗干序列（从值符宫起）：**
```
戊、己、庚、辛、壬、癸、乙、丙、丁（但逆飞）
```

### 暗支查法

暗支是地支的流转，与八门值使相关。

**查法：** 从值使宫对应的地支开始，按阳顺阴逆飞布。

## 三、旺动条件

### 什么是旺动？

**旺：** 用神在旺相宫（能量强）
**动：** 用神被激活（参与生克）

二者缺一不可：旺而不动 → 能量未激活；动而不旺 → 能量弱。

### 旺动判断三要素

1. **宫位旺衰**：看节气，用神是否在旺相宫
2. **生克关系**：是否被生、被克、比和
3. **是否值符**：是否在值符宫（旺动）

### 旺动口诀

> 值符宫中物旺动，比和相生亦为动。
> 若逢休囚又受克，虽有如无万事空。

## 四、六亲定位

六亲：父母、兄弟、妻子、子孙。

### 定位口诀

> 父居西北乾六宫，母居西南坤二宫。
> 妻财居中五宫内，官鬼在艮艮八中。
> 子孙东南巽四宫，兄弟冲对反宫逢。

### 具体对应

| 六亲 | 对应宫 | 对应星 |
|------|--------|--------|
| 父亲 | 乾六宫 | 天心 |
| 母亲 | 坤二宫 | 天芮 |
| 妻子（财） | 中五宫（寄） | 天禽 |
| 官鬼（事业） | 艮八宫 | 天任 |
| 子孙 | 巽四宫 | 天辅 |
| 兄弟 | 对冲宫（与用神对冲） | 对冲星 |

**应用：** 问父母看乾坤，问子女看巽四，问配偶看中五。

## 五、旬空与出空

### 旬空

每6天一个旬，两个地支轮空。

**六旬空表：**
- 甲子旬：戌亥空
- 甲戌旬：申酉空
- 甲申旬：午未空
- 甲午旬：辰巳空
- 甲辰旬：寅卯空
- 甲寅旬：子丑空

### 出空日

空亡地支对应的日期出现时，空亡"出空"。

**例：** 戌亥空，遇戌日或亥日则出空。

## 六、实战案例

### 案例：问事业发展

**起盘：** 时干在震三，开门在巽四

**分析：**
1. 遍干看：开门从值符宫（假设在离九）顺飞而来
2. 暗干：开门宫天盘暗干为丙 → 从离九游来
3. 路径：离九（值符）→ 艮八 → 兑七 → 乾六 → 中五 → 巽四（开门）
4. 含义：从名声（离）→ 止步（艮）→ 口舌（兑）→ 领导（乾）→ 核心（中）→ 开门（巽）
5. 解读：经历"名声→止步→口舌→领导→核心"后，事业在巽四（东南）打开

**结论：** 事业发展路径清晰，但经过波折（艮止、兑口舌），最终得领导（乾）和核心（中）支持，在东南方（巽）有突破。

## 本章小结

**遍干是奇门的"时间维度"**——不只是看当下，还要看流转。

**关键收获：**
1. 三定律：甲随符行、阳顺阴逆、六仪三奇序
2. 暗干暗支：追踪来处去处
