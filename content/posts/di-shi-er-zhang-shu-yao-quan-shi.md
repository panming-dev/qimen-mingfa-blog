看盘容易，看准难。十几个符号挤在一起，到底听谁的？

<p>本章是《遁甲括囊集》原文明文的解盘宪法——七条不可违背的金律：1）格局定框架，门宫定吉凶；2）旺衰定能量，胎没决兴衰；3）用神与时干生克定祸福；4）宫位定领域，星门神定细节；5）多格并看，分其主次；6）时空参断，旺动为凭；7）整体观象，忌执一格。每条金律都配有实战案例和速查口诀。掌握这七条，任何复杂的盘都能在7分钟内理清头绪。</p>

## 一、金律一：格局定框架，门宫定吉凶

### 核心公式

```
最终判断 = 格局象意 × 门宫配合度
```

### 解读

**格局**告诉你"是什么事"（如进茹格=宜前进、正冲格=对抗）。

**八门**告诉你"这件事的状态"：
- 吉门（开、休、生）→ 事情本身状态好
- 凶门（伤、杜、死、惊、景）→ 事情本身有缺陷

### 实战表格

| 格局 | 门状态 | 综合判断 | 例子 |
|------|--------|----------|------|
| 吉格 | 吉门 | **大吉** | 进茹格+开门 → 事业大进 |
| 吉格 | 凶门 | 吉中带疵 | 进茹格+惊门 → 进中有惊 |
| 凶格 | 吉门 | 凶中藏吉 | 正冲格+休门 → 对抗中得休息 |
| 凶格 | 凶门 | **大凶** | 正冲格+死门 → 冲突致命 |

## 二、金律二：旺衰定能量，胎没决兴衰

### 旺衰八状态

| 状态 | 能量 | 适宜 |
|------|------|------|
| **旺** | peak，蓬勃 | 创作、进取 |
| **相** | 次旺，支持 | 建立、巩固 |
| **胎** | 孕育，萌芽 | 安业、培育 |
| **没** | 衰退，沉溺 | 忌渡河、冒险 |
| **死** | 僵化，不活跃 | 忌卜病、行动 |
| **囚** | 被囚，受制约 | 忌词讼、争讼 |
| **休** | 分散，退步 | 宜退步、守旧 |
| **废** | 废弃，失位 | 宜守旧、不创新 |

### 口诀

> 旺喜创作，相喜建立，胎喜安业。
> 没忌渡河，死忌卜病，囚忌词讼，废宜守旧，休宜退步。

**重要：** 格局相同，旺衰不同，结论可能完全相反！

## 三、金律三：用神×时干，生克定祸福

### 核心关系

**时干 = 我（求测者）**
**用神 = 事（所问之事）**

二者关系，决定事情成败。

### 生克判断表

| 关系 | 含义 | 吉凶 |
|------|------|------|
| 时干生用神 | 我付出，支持这件事 | 吉（我主动） |
| 用神生时干 | 事情回报我 | 大吉（事主动） |
| 用神克时干 | 事情控制我、压制我 | 凶（我被制） |
| 时干克用神 | 我控制事情 | 半吉半凶（我能控但费力） |
| 比和 | 朋友、同类 | 平，无吉无凶 |

### 口诀

> 我生之事我费力，事生我来有贵人。
> 我制之事我主动，事制于我受压力。

## 四、金律四：宫位定领域，星门神定细节

### 九宫领域速查

| 宫 | 领域 | 看什么 |
|----|------|--------|
| 坎一 | 智慧、水、险陷 | 学业、智慧 |
| 坤二 | 母亲、土地、顺承 | 母亲、房产 |
| 震三 | 行动、事业起步 | 事业、行动 |
| 巽四 | 进入、贸易、合同 | 合作、贸易 |
| 中五 | 核心、本人 | 自身状态 |
| 乾六 | 领导、父亲、事业 | 领导、事业 |
| 兑七 | 口舌、少女 | 口舌、饮食 |
| 艮八 | 子孙、止步 | 子孙、土地 |
| 离九 | 名声、文书 | 名声、文书 |

### 星门神细化

一个宫是"小宇宙"，星门神干都要看。

**举例：** 开门在震三
- 宫：震三 → 事业起步
- 门：开门 → 事业
- 星：天冲 → 主动、有冲劲
- 组合：合作型的事业起步，需主动但防冲动

## 五、金律五：多格并看，分其主次

一个宫内常有多个格局，必须分主次。

### 主次权重

| 优先级 | 格局类型 | 权重 |
|--------|----------|------|
| 1 | 奇仪组合格（天盘干+地盘干） | 40% |
| 2 | 星门神组合格 | 30% |
| 3 | 特殊状态（空、墓、刑） | 20% |
| 4 | 其他辅助格局 | 10% |

### 案例

宫内：乙+庚（日奇被刑） + 天冲星 + 伤门 + 白虎 + 击刑

**解读：**
1. 主格：乙+庚（40%）→ 合作破裂
2. 星门神：天冲+伤门+白虎（30%）→ 主动出击但受伤
3. 特殊状态：击刑（20%）→ 官非伤病

**综合：** 合作破裂引发冲突，需防伤病官非。

## 六、金律六：时空参断，旺动为凭

### 三个时间维度

| 维度 | 含义 | 看什么 |
|------|------|--------|
| 时干 | 当前时刻的能量 | 事情的当下状态 |
| 节气 | 月份的旺衰 | 各要素的旺衰状态 |
| 年命 | 求测者的八字 | 个人运势共振 |

### 旺动 vs 旺衰

- **旺动**：要素是否被"激活"
- **旺衰**：要素当下的能量强弱

**二者缺一不可：**
- 旺动但休囚 → 有心无力
- 旺衰但不动 → 能量未激活

## 七、金律七：整体观象，忌执一格

### 七维度综合

1. **格局** → 基础含义
2. **门** → 事情的状态
3. **星** → 能量特质
4. **神** → 无形助力/阻力
5. **宫** → 对应领域
6. **旺衰** → 能量强弱
7. **用神关系** → 我与事的关系

**口诀：** 一时十格仔细详，五吉五凶半得失。三格吉兮七格凶，凶多吉少事忧室。

## 八、解盘七步法（Checklist）

解任何一个盘，按这个流程：

### 阶段1：基础定位（1分钟）
- [ ] 找时干宫 → 求测者状态
- [ ] 找用神宫 → 所问事项
- [ ] 找值符宫 → 主导力量、贵人

### 阶段2：格局扫描（2分钟）
- [ ] 看用神宫内，天盘干+地盘干 → 有无格局？
- [ ] 宫内星+门+神 → 有无格局？
- [ ] 宫内特殊状态 → 空？墓？刑？

### 阶段3：关系判断（2分钟）
- [ ] 用神宫与时干宫的生克
- [ ] 用神宫与值符宫的生克
- [ ] 各宫之间的互动

### 阶段4：旺衰校准（1分钟）
- [ ] 用神宫旺衰？
- [ ] 时干宫旺衰？
- [ ] 关键要素是否旺动？

### 阶段5：综合结论（1分钟）
- [ ] 七律综合
- [ ] 吉凶定论
- [ ] 建议方向

**总计：** 7分钟解一局。熟练后3分钟。

## 九、常见错误

### 错误1：格局定一切
**纠正：** 格局只是基础，还要看宫内是否有解神、是否逢空、用神旺衰。

### 错误2：吉门万事吉
**纠正：** 吉门要配合格局、旺衰。

### 错误3：凶门万事凶
**纠正：** 凶门有时正好——惊门主口舌，正适合谈判。

### 错误4：只看用神，不看时干
**纠正：** 时干与用神的生克，才是判断的核心。

## 本章小结

**枢要诠释是解盘的"宪法"**——七条金律必须遵守。

**核心收获：**
1. 格局+门宫 → 基础框架
2. 旺衰 → 能量校准
3. 用神×时干 → 核心关系
4. 多格分主次 → 不执一
5. 七维综合 → 整体观象

**学习建议：**
1. 把七条金律打印，贴在桌前
2. 每次解盘，按Checklist过一遍
3. 先模仿，后创新
