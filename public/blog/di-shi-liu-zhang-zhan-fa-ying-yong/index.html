第十六章：占法应用大全——八大类事项的实战指南

奇门遁甲实战占卜完整手册：事业、财运、婚姻、健康、诉讼、出行、寻物、学业，八大门类一网打尽，含快速查表与案例。
April 28, 2024 · 2 min · 355 words · 磐鸣

学完前面十五章，你已经掌握了奇门遁甲的全部理论。最后一章，我们进入实战——具体事项怎么占？

一、事业占（开门为核心）#

用神体系#

  • 主用神： 开门（事业宫）
  • 辅用神： 时干（你自己）、值符（领导/贵人）
  • 参考： 天心星（管理能力）、日干（自身状态）

判断要点#

  1. 开门宫的状态（旺衰、门星神）
  2. 开门与时干的关系
  3. 开门宫有无吉格（进茹格、三奇游甲）

实战案例#

问： 面试能否成功？

盘： 开门在震三，天冲星，伤门，正冲格

分析：

  • 开门在震三（旺）→ 事业有生机
  • 天冲+伤门 → 主动但易受伤
  • 正冲格 → 有对抗
  • 开门克时干 → 你能控制局面

结论： 能成，但过程有冲突（正冲格+伤门），需防冲动（天冲）。

二、财运占（生门为核心）#

用神体系#

  • 主用神： 生门（财门）
  • 辅用神： 六合（合作）、时干
  • 参考： 甲子戊（资本）、戊+生（资金生财）

判断要点#

  1. 生门宫旺衰（财能量）
  2. 生门生时干（财来助我）还是时干生门（我求财费力）
  3. 六合状态（合作是否顺利）

快速查表#

生门宫位财源类型
震三（东）动中求财、创业
巽四（东南）贸易、信息财
中五核心财、自身财
乾六（西北）事业财、领导财
兑七（西）口舌财、销售财

案例#

问： 投资项目能否盈利？

盘： 生门在巽四，六合在震三，合格格（乙+庚）

分析：

  • 生门在巽四（相）→ 财运中等
  • 六合在震三 → 合作在东方
  • 乙+庚合格 → 合作成
  • 生门生时干 → 财主动来

结论： 可投，合作能成，财主动找上门。

三、婚姻占（六合+乙庚）#

用神体系#

  • 主用神： 六合（婚姻关系）
  • 男用神： 庚（阳干）
  • 女用神： 乙（阴干）
  • 参考： 日干（自身）、时干（状态）

判断要点#

  1. 六合宫旺衰（婚姻能量）
  2. 乙庚关系（是否相克/相生）
  3. 六合与双方的关系

乙庚关系速查#

组合名称象意
乙+庚日奇被刑夫妻不和，合作破裂
庚+乙太白合日男方主动，但阻
乙+丁玉女守门女有外遇
丁+乙星奇入户男有外遇
乙+丙龙入云贵人提携婚姻

案例#

问： 我和TA能结婚吗？

盘： 六合在兑七，乙在震三，庚在坎一

分析：

  • 六合在兑七（胎）→ 婚姻孕育中
  • 乙在震三（旺）→ 女方积极主动
  • 庚在坎一（休）→ 男方状态一般
  • 乙+庚（日奇被刑）→ 有摩擦
  • 六合宫见丁奇 → 有第三方介入

结论： 能成，但女方付出多，男方较被动，且有第三方（可能是家庭、物质）介入。

四、健康占（天芮为核心）#

用神体系#

  • 主用神： 天芮星（疾病星）
  • 参考： 死门（病状）、时干（患者）

判断要点#

  1. 天芮落宫（病位）
  2. 天芮旺衰（病轻重）
  3. 克应（什么能治这个病）

九宫病位对应#

天芮落宫对应器官疾病类型
坎一肾、耳、膀胱水病、寒症
坤二腹、脾胃土病、湿症
震三足、肝、发木病、风症
巽四股、胆、气管木病、气郁
中五脾胃、胰腺土病、中焦
乾六头、肺、大肠金病、燥症
兑七口、舌、肺金病、口疮
艮八手、胃、鼻土病、消化不良
离九眼、心、心脏火病、心脑血管

克应（治疗方向）#

克应原则： 看哪个宫克制天芮宫。

  • 天芮在震三（木）→ 兑七（金）克之 → 西方就医
  • 天芮在坎一（水）→ 离九（火）克之 → 南方就医

五、诉讼占（值符+惊门）#

用神体系#

  • 原告： 值符（起诉方）
  • 被告： 值使（应诉方）
  • 事体： 惊门（口舌、诉讼）

判断要点#

  1. 值符 vs 值使的生克
  2. 惊门状态
  3. 六合（调解可能）

案例#

问： 官司能否赢？

盘： 值符在巽四，值使在震三，惊门在兑七

分析：

  • 值符（巽木）克值使（震木）→ 原告压被告
  • 惊门在兑（金）→ 口舌激烈
  • 六合在乾 → 有调解可能
  • 白虎在震 → 被告有凶象

结论： 原告优势，但过程激烈（惊门+白虎），建议调解（六合在乾）。

六、出行占（开门+马星）#

用神体系#

  • 主用神： 开门（出行）
  • 关键： 马星（驿马星）
  • 参考： 时干（自身状态）

马星查法#

马星在哪个宫，那个方向就动。

马星方位动象
寅午戌马在申西南偏西申宫动
申子辰马在寅东北偏东寅宫动
巳酉丑马在亥西北偏北亥宫动
亥卯未马在巳东南偏南巳宫动

判断要点#

  1. 开门状态（能否出行）
  2. 马星位置（方向）
  3. 开门与马星关系

七、寻物占（伤门+杜门）#

用神体系#

  • 丢失： 伤门（损失）
  • 隐藏： 杜门（隐藏）
  • 盗贼： 玄武（偷窃）

判断要点#

  1. 伤门落宫（方位）
  2. 杜门落宫（是否隐藏）
  3. 玄武落宫（是否被盗）

方位速查#

伤门宫失物方位
坎一北方、水边
坤二西南、田野
震三东方、树上
巽四东南、草丛
乾六西北、高处
兑七西方、金属旁
艮八东北、山上
离九南方、火边

八、学业占（辅星+景门）#

用神体系#

  • 主用神： 辅星（天辅，学业星）
  • 参考： 景门（文书）、丁奇（证书）

判断要点#

  1. 辅星旺衰（学业基础）
  2. 景门状态（考试运）
  3. 时干生扶（自身努力）

快速判断#

辅星宫学业状态
旺相宫学业优秀
休囚宫学业一般
入墓宫发挥受限
空亡宫名落孙山

九、多事同占#

原则： 一事一用神，分开看。

方法：

  1. 事业看开门
  2. 财运看生门
  3. 健康看天芮
  4. 分别分析，不混
