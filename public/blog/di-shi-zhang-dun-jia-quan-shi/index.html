第十章：遁甲诠释——占断与克应全解

克应，就是"这个符号在这个宫，对应什么事"。八门在震三宫是什么事？九神在坎一宫代表什么？

五、占天象#

1. 占风云#

  • 天柱+白虎+惊门 → 狂风大作
  • 巽宫+天辅 → 微风拂面
  • 坤宫+天芮 → 阴云密布

2. 占雷雨#

  • 震宫+天冲+伤门 → 雷声大作
  • 坎宫+天蓬+休门 → 暴雨倾盆

3. 占晴雨#

  • 离宫+天英+景门 → 晴天
  • 坤宫+天芮+死门 → 阴天

六、占国运#

九州分野#

  • 坎宫（北）：河北、辽宁、内蒙
  • 坤宫（西南）：四川、重庆、贵州
  • 震宫（东）：江苏、浙江、上海
  • 巽宫（东南）：广东、福建、海南
  • 乾宫（西北）：陕西、甘肃、宁夏
  • 兑宫（西）：山西、河南、湖北
  • 艮宫（东北）：黑龙江、吉林、辽宁
  • 离宫（南）：湖南、江西、广西

方法： 看值符落宫，对应地域运势。

七、占帝星#

紫微垣#

  • 乾宫+天心+值符 → 帝星明亮
  • 离宫+天英+太阴 → 后星显赫

判断： 帝星旺则国运昌隆，衰则帝星暗淡。

八、实战案例#

案例：占失物#

问： 手机丢失，能否找回？

起盘： 伤门在坎一，玄武在震三

分析：

  1. 伤门（损失）在坎一（水）→ 落在北方或水边
  2. 玄武（盗贼）在震三（东）→ 东边方向
  3. 时干在兑七 → 物在西方

结论： 手机在西北方向（坎一+兑七之间），可寻。

本章速记#

克应核心
八门克应门在何宫，何事发生
九星克应星在何宫，能量特质
九神克应神在何宫，无形助力
十干射覆干象物类，快速判断
占天象星宫组合，风云雷电
占国运值符分野，地域兴衰
占帝星紫微乾离，帝星明暗

本章小结#

克应是奇门占断的"答案库"。 遇到具体事项，查表即可。

关键收获：

  1. 八门克应：门在何宫，对应何事
  2. 九星克应：星在何宫，能量如何
  3. 九神克应：神在何宫，助力阻力
  4. 射覆表：干象物类
  5. 占天象国运：分野对应

学习建议：

  1. 八门九星九神克应表背熟
  2. 遇到占断先查表
  3. 实践中验证准确性
