第二章：飞盘起局——如何排出奇门遁甲盘

奇门遁甲排盘完整教程：从节气、直符、直使到飞布规则，手把手教你排出准确的奇门盘。
January 22, 2024 · 2 min · 418 words · 磐鸣

起局是奇门遁甲的第一步。局起错了，后面全错。飞盘起局法是明代《奇门鸣法》传下的秘法，比常见的"阳盘"更精准，也是本书的核心方法。

这一章，我们用最直白的语言，一步步教你排出奇门盘。不跳步，不简化，从"今天该用哪一局"开始，到"直符直使在哪"，再到"星门怎么飞"，最后检查"暗干暗支对不对"。跟完这一章，你能在任何时间、任何地点，独立起出一张准确的奇门盘。

一、起局三步法总览#

飞盘起局分三步：

第一步：定局（按节气选阳遁/阴遁）
    ↓
第二步：布地盘（把六仪三奇固定布在九宫）
    ↓
第三步：飞天盘（从值符宫顺/逆飞布星门）

每一步都有明确的规则，没有"感觉"或"灵活处理"——奇门是定量的时空模型，每一步都必须算对。

二、第一步：定局——节气与局数的对应#

1. 节气与局数对照表#

奇门用节气定局，每个节气15天，分上中下三元（每元5天）。

阳遁（冬至→夏至）：

节气上元局中元局下元局
冬至阳一局阳七局阳四局
小寒阳二局阳八局阳五局
大寒阳三局阳九局阳六局
立春阳八局阳五局阳二局
雨水阳九局阳六局阳三局
惊蛰阳一局阳七局阳四局
春分阳三局阳九局阳六局
清明阳四局阳一局阳七局
谷雨阳五局阳二局阳八局
立夏阳四局阳一局阳七局
小满阳五局阳二局阳八局
芒种阳六局阳三局阳九局

阴遁（夏至→冬至）：

节气上元局中元局下元局
夏至阴九局阴三局阴六局
小暑阴八局阴二局阴五局
大暑阴七局阴一局阴四局
立秋阴二局阴五局阴八局
处暑阴一局阴四局阴七局
白露阴三局阴六局阴九局
秋分阴七局阴一局阴四局
寒露阴六局阴九局阴三局
霜降阴五局阴八局阴二局
立冬阴六局阴九局阴三局
小雪阴五局阴八局阴二局
大雪阴四局阴七局阴一局

速记口诀：

冬至一七四，小寒二八五。 大寒春分三九六，雨水九六三。 夏至九三六，小暑八二五。 大暑秋分七一四，立秋二五八。 寒露立冬六九三，处暑一四七。 霜降小雪五八二，大雪四一七。

口诀含义：每个节气对应的三个元（上中下）的局数。比如"冬至一七四" = 冬至上元阳一局、中元阳七局、下元阳四局。

2. 如何确定当前用哪一局？#

步骤：

  1. 确定问事时间对应的节气（不是农历节气，是二十四节气）
  2. 确定该节气内的第几天（1-5上元，6-10中元，11-15下元）
  3. 查表得局数

举例： 2024年2月4日立春

  • 节气：立春
  • 时间：2月4日，属于上元（1-5日）
  • 查表：立春上元 → 阳八局
  • 结论：用阳八局起盘

三、第二步：布地盘——六仪三奇的固定布局#

地盘是固定不变的，不随时间改变。每个局的地盘布局是固定的。

1. 阳遁地盘布局规则#

阳遁：从坎一宫（北方）起甲子戊，顺时针排布六仪三奇。

阳遁一局示例：

坎一（北）：甲子戊
坤二（西南）：甲戌己
震三（东）：甲申庚
巽四（东南）：甲午辛
中五（中央）：甲辰壬（寄）
乾六（西北）：甲卯乙
兑七（西）：甲寅癸
艮八（东北）：空
离九（南）：空

飞布口诀：

子戊、戌己、申庚、午辛、辰壬、卯乙、寅癸，顺飞九宫。

2. 阴遁地盘布局规则#

阴遁：从离九宫（南方）起甲子戊，逆时针排布六仪三奇。

阴遁一局（实际是阴九局）示例：

离九（南）：甲子戊（起）
艮八（东北）：甲戌己
兑七（西）：甲申庚
乾六（西北）：甲午辛
中五（中央）：甲辰壬（寄）
巽四（东南）：甲卯乙
震三（东）：甲寅癸
坤二（西南）：空
坎一（北）：空

飞布口诀：

子戊、戌己、申庚、午辛、辰壬、卯乙、寅癸，逆飞九宫。

3. 寄宫问题#

中五宫（中央）没有对应的门。地盘六仪中，有一个要"寄"到中五宫。

寄法规则： 按阴阳遁，寄在值符前一位。

  • 阳遁：寄在乾六前一位 → 中五
  • 阴遁：寄在巽四前一位 → 中五

四、第三步：飞天盘——值符值使的顺逆飞布#

天盘是动态的，随起局时间变化。核心是找值符和值使。

1. 找旬首#

根据起局时间的天干地支，找出当前属于哪个"旬"。

旬的规则： 每6天一个循环，以"甲"开头。

  • 甲子旬：甲子、乙丑、丙寅、丁卯、戊辰、己巳
  • 甲戌旬：甲戌、乙亥、丙子、丁丑、戊寅、己卯
  • 甲申旬：甲申、乙酉、丙戌、丁亥、戊子、己丑
  • 甲午旬：甲午、乙未、丙申、丁酉、戊戌、己亥
  • 甲辰旬：甲辰、乙巳、丙午、丁未、戊申、己酉
  • 甲寅旬：甲寅、乙卯、丙辰、丁巳、戊午、己未

口诀： 甲己还生甲，乙庚丙作初，丙辛从戊起，丁壬庚子居，戊癸何方发，壬子是真途。

2. 定值符值使#

值符 = 值班的九星 值使 = 值班的八门

找法：

  1. 确定旬首的天干（如甲子→甲子戊）
  2. 看这个天干在地盘哪个宫 → 该宫对应的星就是值符
  3. 该宫对应的门就是值使

星门对应表：

  • 坎一宫：天蓬 + 休门
  • 坤二宫：天芮 + 死门
  • 震三宫：天冲 + 伤门
  • 巽四宫：天辅 + 杜门
  • 中五宫：天禽 + 死门（寄）
  • 乾六宫：天心 + 开门
  • 兑七宫：天柱 + 惊门
  • 艮八宫：天任 + 生门
  • 离九宫：天英 + 景门

3. 飞布星门#

规则：

  1. 值符落哪宫，就从哪宫开始飞
  2. 阳遁顺时针飞，阴遁逆时针飞
  3. 飞布顺序：值符 → 值使 → 其余八星/八门

阳遁顺飞示例（阳一局，值符在坎一）：

坎一（起）→ 坤二 → 震三 → 巽四 → 中五 → 乾六 → 兑七 → 艮八 → 离九

阴遁逆飞示例（阴九局，值符在离九）：

离九（起）→ 艮八 → 兑七 → 乾六 → 中五 → 巽四 → 震三 → 坤二 → 坎一

4. 暗干暗支#

飞布时，每个宫还有天盘暗干和天盘暗支——这是隐藏的流转信息，用于"周游章"的分析。

暗干规则： 随值符飞布的天干序列。 暗支规则： 随值使飞布的地支序列。

五、完整起例：阳四局，壬午时#

我们完整走一遍起局流程。

1. 定局#

  • 节气：立夏（5月5-6日）
  • 时间：立夏上元（1-5日）
  • 查表：立夏上元 → 阳四局

2. 布地盘（阳四局固定布局）#

坎一：甲午辛
坤二：甲辰壬
震三：甲卯乙
巽四：甲寅癸
中五：甲子戊（寄）
乾六：甲戌己
兑七：甲申庚
艮八：空
离九：空

3. 找旬首和值符值使#

  • 时干：壬午时
  • 旬首：甲辰（壬午属于甲辰旬）
  • 甲辰 → 地盘在震三宫
  • 震三宫对应的星：天冲 → 值符
  • 震三宫对应的门：伤门 → 值使

4. 飞天盘#

  • 值符（天冲）在震三（起宫），阳四局阳遁，顺时针飞： 震三（天冲）→ 巽四（天辅）→ 中五（天禽）→ 乾六（天心）→ 兑七（天柱）→ 艮八（天任）→ 离九（天英）→ 坎一（天蓬）→ 坤二（天芮）

  • 值使（伤门）在震三（起宫），顺时针飞： 震三（伤）→ 巽四（杜）→ 中五（死）→ 乾六（开）→ 兑七（惊）→ 艮八（生）→ 离九（景）→ 坎一（休）→ 坤二（休？实际顺序按门序）

5. 排三奇#

三奇（乙丙丁）按固定顺序排在六仪之后：

  • 阳遁：六仪顺序为戊己庚辛壬癸，三奇为乙丙丁
  • 布在值符宫之后的相应位置

六、常见错误与检查#

错误1：节气搞错#

表现： 用了错误的节气，局数全错。

检查： 对照节气表，确认当前节气及其上中下元。

错误2：阳顺阴逆搞反#

表现： 阴遁也顺飞。

检查： 记住口诀：阳遁阳顺，阴遁阴逆。

错误3：寄宫放错#

表现： 中五宫六仪位置错。

检查： 中五永远寄在值符前一宫。

错误4：值符值使找错#

表现： 旬首算错，导致值符值使全错。

检查： 重新算旬首，对照地盘确认。

本章速记表#

步骤关键动作检查点
1. 定局按节气查表阳遁/阴遁是否正确
2. 布地盘按局数布六仪六仪顺序是否正确
3. 找旬首按时干查旬旬首天干是否正确
4. 定值符旬首天干在地盘宫星门对应是否正确
5. 飞天盘阳顺阴逆飞布方向是否正确

本章小结#

飞盘起局是奇门解盘的基础。 局起不对，后面全是瞎蒙。

关键要点：

  1. 节气定局：冬至后阳遁，夏至后阴遁
  2. 地盘固定：每个局的六仪布局是固定的，背熟
  3. 值符值使：旬首决定，是飞布的起点
  4. 阳顺阴逆：飞布方向的核心规则

学习建议：

  1. 把节气-局数对照表打印贴在桌前
  2. 每天起一局练手，连续21天形成肌肉记忆
  3. 用软件（如金亮奇门）对照检查自己的起盘
