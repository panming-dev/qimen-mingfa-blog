遍干

第十一章：遍干——天干流转的轨迹追踪

遍干，就是天干在九宫中的流转轨迹。知道天干从哪里来、到哪里去，才能看出事物的发展脉络。 一、遍干三定律 1. 甲随符行 定律： 甲干（及六仪三奇）的流转，永远跟随值符。 含义： 值符走到哪，甲（及六仪三奇）就跟到哪。这是飞盘的基础。 例： 阳四局，值符在震三，则甲子戊从震三开始顺飞。 2. 阳顺阴逆 定律： 阳遁顺飞，阴遁逆飞。 含义： 能量流动方向由阴阳决定。 口诀： 阳顺阴逆别混淆，夏至之后用阴遁。 3. 六仪三奇序 定律： 飞布顺序固定： 六仪：戊 → 己 → 庚 → 辛 → 壬 → 癸 三奇：乙 → 丙 → 丁 含义： 无论从哪个宫起，顺序不变。 二、暗干暗支 什么是暗干暗支？ 每个宫除了表面的天盘干、地盘干，还有隐藏的流转信息： 天盘暗干：天盘干从哪宫来？ 地盘暗支：地盘支向哪宫去？ 暗干查法 暗干就是值符飞布时携带的天干序列。 阳遁暗干序列（从值符宫起）： 戊、己、庚、辛、壬、癸、乙、丙、丁 阴遁暗干序列（从值符宫起）： 戊、己、庚、辛、壬、癸、乙、丙、丁（但逆飞） 暗支查法 暗支是地支的流转，与八门值使相关。 查法： 从值使宫对应的地支开始，按阳顺阴逆飞布。 三、旺动条件 什么是旺动？ 旺： 用神在旺相宫（能量强） 动： 用神被激活（参与生克） 二者缺一不可：旺而不动 → 能量未激活；动而不旺 → 能量弱。 旺动判断三要素 宫位旺衰：看节气，用神是否在旺相宫 生克关系：是否被生、被克、比和 是否值符：是否在值符宫（旺动） 旺动口诀 值符宫中物旺动，比和相生亦为动。 若逢休囚又受克，虽有如无万事空。 ...
