轨迹追踪

第十三章：周游——追踪事物的发展轨迹

周游，就是天干在九宫中的流转过程。从用神宫的"天盘暗干"，可以看出它从哪个宫游来（来处）；从"旺衰推演"，可以看出它将游向哪个宫（去处）。 一、周游的核心 周游的本质，是天干（六仪三奇）在九宫中的流转路径。 关键概念 本位：每个干在九宫中的"原始位置" 游处：当前所在宫 来处：从哪个宫游过来 去处：将要游向哪个宫 最简单的方法：看暗干支 现代软件会在每个宫显示天盘暗干——那是隐藏的流转信息。 初学者记住： 看用神宫的天盘暗干 → 知道它从哪里来 看用神宫的地盘暗干 → 知道它要到哪里去 二、怎么看"来处"？ 方法1：从旬首推算 步骤： 找到当前旬的旬首（甲子、甲戌、甲申…） 旬首落在哪个宫？那就是起点 从起点，按阳顺阴逆，数到用神宫 → 中间经过的宫就是"来处路径" 举例： 阳四局，壬午时 旬首：甲辰（壬午属于甲辰旬） 甲辰落震三宫（按阳四局排） 用神宫在离九 阳顺数：震三→巽四→中五→乾六→兑七→艮八→离九 路径： 震→巽→中→乾→兑→艮→离 解读： 这件事从震宫（行动）开始，经过巽（进入）、中（核心）、乾（领导）、兑（口舌）、艮（止静），最终到离（名声）。 方法2：看暗干游变 软件显示天盘暗干。暗干的排布就是流转路径。 口诀： 暗干流转处，来去有根源。 三、怎么看"去处"？ 方法：按旺衰推演 事物的发展，遵循旺→相→胎→没→死→囚→休→废的循环。 步骤： 看用神宫当前状态（旺衰） 按顺时针方向，下一个状态是什么？ 对应的宫位就是"去处" 举例： 用神宫在震三，当前状态旺 旺 → 相（巽四宫） 相 → 胎（中五宫） 胎 → 没（乾六宫） 路径： 震（旺）→ 巽（相）→ 中（胎）→ 乾（没） 解读： 当前：蓬勃发展的旺状态（震） 下一步：稳定支持的相状态（巽） 再下一步：孕育萌发的胎状态（中） 未来：开始衰退的没状态（乾） 建议： 当前宜大进，下一步宜稳，再下一步宜守，未来需警惕。 ...
