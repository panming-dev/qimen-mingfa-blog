第五章：鸣法格局——奇门遁甲的判断公式

格局是奇门遁甲的判断公式。看到某个组合，就知道这件事大概是什么走向。 比如"进茹格"——三个门依次顺排，代表宜前进、步步高升；“正冲格”——门冲对宫，代表对抗、冲突。本章系统讲解《奇门鸣法》中的核心格局：从最基础的进茹退茹，到冲合格局，再到特殊状态（入墓、旬空、击刑）。每个格局都配有实际案例，告诉你"什么时候这个格局准，什么时候要灵活变通"。 一、什么是格局？ 格局，就是特定符号组合形成的固定判断模式。 就像下棋的"杀招"——看到这个布局，就知道要这样走。奇门也一样： 看到"三进宫" → 进茹格 → 宜前进 看到"门冲对宫" → 正冲格 → 有对抗 看到"乙+庚" → 日奇被刑 → 合作破裂 口诀： 一时十格仔细详，五吉五凶半得失。三格吉兮七格凶，凶多吉少事忧室。 二、鸣法核心十三格 1. 进茹格（最吉） 条件： 八门顺排（开→休→生→伤→杜→景→死→惊→开，顺时针）。 象意： 步步高升，运势上扬，宜进取。 应用： 创业、求职、投资、开拓。 案例： 问求职，开门在震三，休门在巽四，生门在中五——三进，大吉。 2. 退茹格（宜守） 条件： 八门逆排（开→惊→死→景→杜→伤→生→休→开，逆时针）。 象意： 步步后退，宜守不宜攻。 应用： 守成、避险、撤退、止损。 口诀： 进茹宜前进，退茹宜退守。莫要强出头，顺势而为吉。 3. 正冲格（对抗） 条件： 用神宫与对宫（对冲）的门/星/干有冲。 象意： 对抗、冲突、争执。 应用： 诉讼、竞争、谈判。 注意： 冲也有冲开的可能，需看旺衰。 4. 合格格（五合） 条件： 天盘干与地盘干有合（甲己合、乙庚合等）。 类型： 合土格： 戊+己、己+戊（土合） 合金格： 庚+辛、辛+庚（金合） 合水格： 壬+癸、癸+壬（水合） 合木格： 甲+乙、乙+甲（木合） 合火格： 丙+丁、丁+丙（火合） 象意： 合作、结合、缘分。 ...
