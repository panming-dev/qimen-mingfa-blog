第十六章：占法应用大全——八大类事项的实战指南

学完前面十五章，你已经掌握了奇门遁甲的全部理论。最后一章，我们进入实战——具体事项怎么占？ 一、事业占（开门为核心） 用神体系 主用神： 开门（事业宫） 辅用神： 时干（你自己）、值符（领导/贵人） 参考： 天心星（管理能力）、日干（自身状态） 判断要点 开门宫的状态（旺衰、门星神） 开门与时干的关系 开门宫有无吉格（进茹格、三奇游甲） 实战案例 问： 面试能否成功？ 盘： 开门在震三，天冲星，伤门，正冲格 分析： 开门在震三（旺）→ 事业有生机 天冲+伤门 → 主动但易受伤 正冲格 → 有对抗 开门克时干 → 你能控制局面 结论： 能成，但过程有冲突（正冲格+伤门），需防冲动（天冲）。 二、财运占（生门为核心） 用神体系 主用神： 生门（财门） 辅用神： 六合（合作）、时干 参考： 甲子戊（资本）、戊+生（资金生财） 判断要点 生门宫旺衰（财能量） 生门生时干（财来助我）还是时干生门（我求财费力） 六合状态（合作是否顺利） 快速查表 生门宫位 财源类型 震三（东） 动中求财、创业 巽四（东南） 贸易、信息财 中五 核心财、自身财 乾六（西北） 事业财、领导财 兑七（西） 口舌财、销售财 案例 问： 投资项目能否盈利？ 盘： 生门在巽四，六合在震三，合格格（乙+庚） 分析： 生门在巽四（相）→ 财运中等 六合在震三 → 合作在东方 乙+庚合格 → 合作成 生门生时干 → 财主动来 结论： 可投，合作能成，财主动找上门。 ...
